# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please prioritize responsible disclosure.

**Please do not open a public issue.**

Instead, please email `udayphlk@gmail.com` with details of the vulnerability. We will address it as soon as possible.

## Supported Versions

| Version | Supported          |
| :------ | :----------------- |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |
